# Chess from Scratch

The goal of this project is to implement chess from scratch while using as little outside code as possible. 

## Rules

What I can use 
- C++ standard library
- Graphics libraries
- Libraries unrelated to chess
- Non-AI assistance
    - stack overflow
    - reddit posts
    - public information about chess algorithms

What I can't use
- Chess bots made by other people
- Chess evaluation tools made by other people
- AI to generate code - conceptual questions and debugging only

## Plans

The goal is to make an CLI chess game with more advanced features like an evaluation engine and bots to play against, especially bots that are pretty difficult. 



## Why this is difficult

Coding the basic rules for chess itself is not particularly hard, though it is tedious. The real challenge, I think, should come from the fact that I am going to have to implement an evaluation tool and chess bots to play against.
